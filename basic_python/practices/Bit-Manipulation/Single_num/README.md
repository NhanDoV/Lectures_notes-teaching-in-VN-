## Description
### Problem
- You are given a non-empty array of integers nums. **Every integer appears twice except for one**.

- Return the integer that appears only once.

-----
### Examples
#### EX 1.
```shell
Input: nums = [3,2,3]

Output: 2 
```

#### EX 2.
```shell
Input: nums = [7,6,6,7,8]

Output: 8 
```
-----
#### Constraints
- `1 <= nums.length <= 10000`
- `10000 <= nums[i] <= 10000`
